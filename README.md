#  Netflix EDA Project

A beginner-friendly Exploratory Data Analysis (EDA) on **Netflix Movies and TV Shows** using Python.

##  Overview
This project analyzes the Netflix dataset using simple EDA techniques. It includes five visualizations:

- Movies vs TV Shows
- Top Countries with Most Titles
- Most Common Content Ratings
- Movie Duration Distribution
- Top 10 Genres

Each graph includes a short insight for easier understanding.

##  Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

##  How to Run
1. Download or open the dataset (`netflix_titles.csv`).
2. Open and run `netflix_eda.ipynb` in Jupyter Notebook.

##  Key Insights
- Netflix has more **Movies** than **TV Shows**.
- The **United States** produces the most titles.
